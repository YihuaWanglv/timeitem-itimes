Project start

我读到《奇特的一生》这本书时，感触颇深。柳比歇夫时间管理法，尽管以前就接触了解过，然而直到我真正阅读完《奇特的一生》这本书的时候，我才对柳比歇夫以及他的时间记录有了更深的理解。同时，我萌生一个想法，参照柳比歇夫的时间记录方式，做一个类似的应用，这个应用应该是同时覆盖各个平台的app。于是我便马上启动了这个项目。

项目规划

1.第一步，TimeItem将上线web pc端应用

2.第二步，TimeItem将增加生成报表，并提供下载等功能

3.第三步，TimeItem将上线Android app

4.第四步，TimeItem将上线iOs app
